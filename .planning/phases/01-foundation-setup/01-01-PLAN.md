<task type="auto">
  <name>Task 3: Create base layout and mobile-first UI components</name>
  <files>src/app/layout.tsx, src/app/page.tsx, src/components/layout/Container.tsx, src/components/ui/Button.tsx</files>
  <action>
    **layout.tsx** — Update the root layout to wire in fonts and set metadata:
    ```tsx
    import type { Metadata } from 'next';
    import { inter, poppins } from '@/lib/fonts';
    import './globals.css';

    export const metadata: Metadata = {
      title: "O'Fallon Bombers Baseball",
      description: "O'Fallon Bombers select youth baseball club — teams, rosters, events, and registration for ages 7U-15U.",
    };

    export default function RootLayout({ children }: { children: React.ReactNode }) {
      return (
        <html lang="en" className={`${inter.variable} ${poppins.variable}`}>
          <body className="min-h-screen bg-white text-navy-900 antialiased">
            {children}
          </body>
        </html>
      );
    }
    ```

    The html element must have both font CSS variable classes. Body uses min-h-screen and antialiased for clean rendering. No dark mode classes.

    **Container.tsx** — Mobile-first centered container:
    ```tsx
    export default function Container({ children, className = '' }: { children: React.ReactNode; className?: string }) {
      return (
        <div className={`px-4 py-6 md:px-6 md:py-8 lg:px-8 lg:py-12 max-w-7xl mx-auto ${className}`}>
          {children}
        </div>
      );
    }
    ```

    Mobile-first: default styles are for phones (compact padding), md: for tablet, lg: for desktop. max-w-7xl centers content on large screens.

    **Button.tsx** — Branded button with primary (yellow) and secondary (navy) variants:
    ```tsx
    export default function Button({
      children,
      variant = 'primary',
      className = '',
      ...props
    }: {
      children: React.ReactNode;
      variant?: 'primary' | 'secondary';
      className?: string;
    } & React.ButtonHTMLAttributes<HTMLButtonElement>) {
      const base = 'inline-flex items-center justify-center px-6 py-3 rounded-lg font-semibold transition-colors duration-200 active:scale-95';
      const variants = {
        primary: 'bg-bombers-yellow text-bombers-navy hover:bg-yellow-600',
        secondary: 'bg-bombers-navy text-white hover:bg-navy-600',
      };
      return (
        <button className={`${base} ${variants[variant]} ${className}`} {...props}>
          {children}
        </button>
      );
    }
    ```

    Buttons have 44px+ tap targets (py-3 = 12px top + 12px bottom + text height), rounded-lg for friendly feel, active:scale-95 for tactile feedback. Yellow primary per user decision (eye-catching CTAs), navy secondary.
